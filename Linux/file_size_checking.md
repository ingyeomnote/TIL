리눅스 서버에서 파일 용량 확인 명령어 du

du는 disk usage의 약자로 디렉토리(폴더)와 파일의 용량을 출력해주는 명령어이다.

>du [옵션] [위치 또는 파일]

디렉토리와 모든 하위 디렉토리의 용량을 표시해준다.
다음과 같이 명령하면 etc 디렉토리와 그 하위 디렉토리의 사용량이 출력된다. 단위는 kbyte이다.
